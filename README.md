## Useful Tool For Monitor CPU Stat

## GetStarted

### Step 1 clone this repo

```bash
git clone https://github.com/ooopSnake/pressure_watcher
```

### Step 2 build

```bash
./build_all.sh
```

### Step 3 execute

```bash
sudo ./pressure_watcher_arm
```
